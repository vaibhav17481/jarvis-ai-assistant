# Jarvis AI Assistant

A voice-controlled virtual assistant built using Python that can perform everyday tasks like opening websites, playing music, telling time, and more — inspired by JARVIS from Iron Man.

---

## Features

*  Voice command recognition
*  Text-to-speech responses
*  Open websites (Google, YouTube, Facebook, Instagram)
*  Play songs directly from YouTube
*  Tell current time
*  Tell current date
*  Open system applications (Notepad, Chrome)
*  Basic command handling

---

## Technologies Used

* Python 3.11
* SpeechRecognition
* gTTS (Google Text-to-Speech)
* pygame
* pywhatkit
* sounddevice
* scipy

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/vaibhav17481/jarvis-ai-assistant.git
cd jarvis-ai-assistant
```

2. Install dependencies:

```bash
pip install pyttsx3 gtts pygame pywhatkit SpeechRecognition sounddevice scipy
```

---

##  Usage

Run the program:

```bash
python main.py
```

Then speak commands like:

* "Open Google"
* "Play Believer"
* "What is the time"
* "Open Notepad"
* "Open Chrome"

---

## Requirements

* Internet connection (for voice recognition & speech)
* Microphone access enabled
* Speakers or headphones for audio output
